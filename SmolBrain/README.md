# SmolBrain Zero Dollar Purchase PoC

## How to run

Set up the environment variable for $MORALIS_KEY

```
$  git clone https://github.com/blocksecteam/defi_poc
$  export $MORALIS_KEY = xxxxx
$  cd ./SmolBrain
$  npm install

then 

$  sh run.sh
```
